     pi-switch：复刻 cc-switch-cli 的交互式 TUI（仅支持 pi agent）

     Context

     pi-switch 是 cc-switch-cli 的复刻项目（napi-rs：Rust cdylib + Node
     bin），管理 pi agent（~/.pi/agent/models.json）的 provider
     profiles。业务层（config/presets/daemon/stats）已完备，但现有
     src-rust/tui.rs（341 行）只是只读浏览。目标：把 cc-switch-cli 的完整 TUI
     观感与交互复刻过来——Dracula 主题、header+左导航+内容区+footer
     布局、Profiles 表格 CRUD、表单（模板 chips + 字段表 + JSON
     预览）、确认弹窗、Toast、帮助 overlay、/ 过滤。

     依赖已就绪：ratatui 0.30 + crossterm 0.29（与 cc-switch 同版本），无需改
     Cargo.toml。

     已确认需求（用户选定）：
     - 导航页全集：Home / Profiles / Presets / Proxy / Stats / Backups / Exit
     - 完整交互复刻（含 / 过滤、? 帮助、详情页、readline 编辑键）
     - 表单 models 字段用逗号分隔单行编辑（保存拆为 ModelEntry
     数组，其余参数取默认值）
